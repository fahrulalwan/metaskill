# Contributing

## Ways to Help

- **Test v1.0** and report what breaks
- **Suggest learning patterns** — what mistakes does your agent keep repeating?
- **Improve the eval scoring** — current formula is basic
- **Build v1.1** — LLM-powered extraction (see `scripts/llm_extract.py`)

## Pull Requests

1. Fork the repo
2. Create a branch: `git checkout -b feat/your-feature`
3. Make your changes
4. Open a PR with context on what you changed and why

## Philosophy

v1.0 is intentionally basic. A rough system you actually run beats a perfect system that's too complex to use. Keep that in mind before adding complexity.
